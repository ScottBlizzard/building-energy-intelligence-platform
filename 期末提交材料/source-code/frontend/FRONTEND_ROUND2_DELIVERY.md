## 技术实现细节

### 图表组件统一接口
```vue
<TrendChart
  :data="analytics.timeSummary"      # 接口返回的真实数据
  :loading="loading.analytics"        # 加载状态
  :error="errors.analytics"           # 错误状态
/>
```

### 引用卡片使用方式
```vue
<CitationCard 
  :citation="citation"                # 引用对象
  :compact="false"                    # 是否紧凑模式
/>
```

### 字段名一致性检查
所有字段名都严格保持与后端接口一致：
- `electricity_kwh` - 电耗数据
- `hvac_kwh` - 空调电耗
- `average_cop` - 平均COP值
- `anomaly_reason` - 异常原因
- `building_name` - 建筑名称
