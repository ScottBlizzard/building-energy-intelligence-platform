# Frontend 说明

前端基于 Vue 3 + Vite，当前已经完成第一次整合，目标是提供适合课程演示和多人协作的多模块工作台。

## 当前模块

- 总览首页
- 数据浏览区
- 统计分析区
- 智能问答区

## 当前已接入的实际能力

- 总览页读取真实 KPI 与数据元信息
- 数据浏览支持建筑筛选、时间范围筛选和 CSV 导出
- 统计分析页读取趋势、建筑对比、COP 排名、异常明细和异常原因统计
- 智能问答区支持结构化回答、知识来源展示和继续追问

## 开发原则

- 先保证接口可接通，再做样式细化
- 避免随意修改接口字段名
- 页面组件尽量职责单一，方便多人协作
- 图表库接入前，先用轻量占位组件保证结构稳定
- 所有数据展示优先围绕真实接口返回结果展开
