# Backend 说明

后端基于 FastAPI，负责样例数据读取、查询、统计分析和问答接口封装。

## 当前目录

- `app/api/routes/`：对外接口
- `app/services/`：数据读取、分析、问答逻辑
- `app/core/`：配置
- `tests/`：接口级测试占位

## 当前已实现能力

- 健康检查
- 数据概览
- 建筑列表与数据元信息
- 原始记录查询
- 时间汇总、建筑对比、COP 排名、异常分析
- 楼层区域分析、设备运行监测、异常工单和运营优化建议
- 原始记录 CSV 导出
- 规则问答与知识文件引用
- 可选外部大模型增强问答，默认关闭，失败时回退本地问答

## 后续建议重点

- 接入真实传感器或数据库时，优先保持当前 API 字段兼容
- 将问答升级为更完整的检索增强生成链路
- 增加认证、日志和更细粒度错误处理
- 持续完善测试覆盖
