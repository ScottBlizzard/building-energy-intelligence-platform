# 实验结果替换清单

在把模板发给其他三个人之前，先按这个清单过一遍。

## 1. 小组统一结果是否已经冻结

- [ ] 主测量方法已经确定
- [ ] 验证方法已经确定
- [ ] 系统边界已经确定
- [ ] 计入和不计入项已经确定
- [ ] 数据功能统计表已经确定
- [ ] 事务功能统计表已经确定
- [ ] UFP / FP 最终数字已经确定

## 2. 报告模板里的公共部分是否已经替换

- [ ] `测量方法` 已替换成真实方法
- [ ] `系统边界与测量范围` 已替换成真实边界
- [ ] 数据功能表已填真实内容
- [ ] 事务功能表已填真实内容
- [ ] UFP / FP / VAF 已填真实结果
- [ ] `结果分析` 已按真实实验结果改写

## 3. 发给每位同学前要提醒他们改的内容

- [ ] 学号
- [ ] 姓名
- [ ] PDF 文件名
- [ ] `个人贡献说明`
- [ ] `实验学习体会`

## 4. 最后导出前再查一次

- [ ] 文中没有残留 `【待替换】`
- [ ] 文中没有残留 `【个人填写】`
- [ ] 四个人最终统计结果一致
- [ ] 四个人个人体会不完全一样
- [ ] 格式已经整理到“同济大学实验报告纸”版式中
- [ ] 导出文件为 PDF
