# 实验二资料包

这个文件夹用于完成老师中途布置的“实验二：软件规模度量”报告。

当前已经准备好的内容：

- [assignment.md](/d:/软件经济管理/实验二/assignment.md)：Canvas 作业说明
- `Experiment Instruction+Exp 2_Software Size Measurement.pdf`：实验指导书
- [01-实验报告模板.md](/d:/软件经济管理/实验二/01-实验报告模板.md)：四个人都可复用的完整报告模板
- [02-FP测量工作表.md](/d:/软件经济管理/实验二/02-FP测量工作表.md)：小组统一填写的功能点测量工作表
- [03-四人分工与个人替换说明.md](/d:/软件经济管理/实验二/03-四人分工与个人替换说明.md)：四个人分别做什么、各自要改哪里
- [04-实验结果替换清单.md](/d:/软件经济管理/实验二/04-实验结果替换清单.md)：正式导出 PDF 前的检查清单

推荐使用顺序：

1. 先看 [assignment.md](/d:/软件经济管理/实验二/assignment.md) 和实验指导书，确认老师要求。
2. 四个人共同完成 [02-FP测量工作表.md](/d:/软件经济管理/实验二/02-FP测量工作表.md)，把系统边界、功能分类和 FP 统计结果定下来。
3. 由你把测量结果填回 [01-实验报告模板.md](/d:/软件经济管理/实验二/01-实验报告模板.md) 的公共部分。
4. 每个人复制一份模板，按 [03-四人分工与个人替换说明.md](/d:/软件经济管理/实验二/03-四人分工与个人替换说明.md) 改自己的姓名、贡献说明和实验体会。
5. 最后按 [04-实验结果替换清单.md](/d:/软件经济管理/实验二/04-实验结果替换清单.md) 检查，再转成 PDF。

注意两点：

- `assignment.md` 明确写了“实验报告内容不必手写”，所以当前按电子版 PDF 准备是对的。
- 四个人的报告主体内容可以高度一致，但不要四份一字不差；至少“个人分工说明”和“学习体会”要各自改写。
